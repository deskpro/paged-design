Book CSS

Customisable themes for book design with CSS

Are you creating books with HTML and CSS? CSS files for books can be long and complicated to write. Our themes give you a head start: pick a theme for your book, add it to your project, and customise it to suit your needs.

Themes

  • Beatrix: named for Ms Potter, a classic children's book theme for simple stories. Goes well with Alice in Wonderland. Download
  • Density: a space-age look for big books. Goes well with The Future of Ideas by Lawrence Lessig. Download
  • Plastered: a wild and wonky look for grabbing attention. Goes well with Little Brother by Cory Doctorow. Download
  • Template: a plain template with very few opinions, and the starting place for creating brand new themes. Goes well with our typography samples. Download

These are just a small start. If you're interested in more themes, tell us so that we know it's worth investing time in more theme development.

How to use a theme

Setting up

  1. You'll need to have NodeJS installed on your computer. Our themes are written in Sass, and you'll use Node to build that into your finished CSS.
  2. Download a theme package and unzip it.
  3. Place the unzipped book-css package in your project. We're assuming that your project is contained in a single folder (like a Git repo).
  4. In the head of your HTML (the content of your book), link to the compiled CSS file in your new theme folder. The path will depend on your folder structure, for example ../book-css/css/themes/beatrix/main.css:
    <link rel="stylesheet" type="text/css" href="../book-css/css/themes/beatrix/main.css">
  5. In the head of your HTML, link to the paged.js script, which will lay out your pages in Chrome. For instance, if you want to always link to the latest official version: 
    <script src="https://unpkg.com/pagedjs/dist/paged.polyfill.js"></script>
  6. Serve your HTML. For example, if you're using Node and have http-server installed, you can run npm run http-server -p 5000 in your project root folder, and visit your site at http://localhost:5000.
Now that you're up and running, you can customise your theme.

Customising a theme

Once you're set up:

  1. Install dependencies for book-css. From the command line, in the book-css folder, enter:

    npm install

    which will install a few modules to node_modules that we need in that folder.

  2. Start our Node script to watch for changes to your styles. Still in the book-css folder at the command line, run:

    npm buildcss

    Node Sass will now the wathcing for changes. Every time you change a file in build-css, your compiled CSS will automatically update and you only need to refresh your browser to see your new styles.

  3. Set your selectors for book features in book-css/themes/YOURTHEME/_selectors.scss (where YOURTHEME is the name of the theme youâ€™ve downloaded).
  4. Set any new values for design variables in book-css/themes/YOURTHEME/_variables.scss. See the template for variables you can set.
  5. Add your own Sass/CSS styles to book-css/themes/YOURTHEME/_styles.scss.

For more detail, see the technical docs.
